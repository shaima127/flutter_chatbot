# WhatsApp Flutter Tutor Bot

بوت واتساب لتعليم لغة Flutter باستخدام الذكاء الاصطناعي (Groq) وتقنية RAG.

## المميزات
- **نظام طلاب متكامل:** تخزين بيانات الطلاب ونقاطهم ومستواهم.
- **دروس مرتبة:** تقديم الدروس بناءً على `order_index`.
- **اختبارات قصيرة:** نظام كويزات بعد كل درس لزيادة النقاط.
- **ذكاء اصطناعي (Groq):** الإجابة على استفسارات الطلاب البرمجية.
- **RAG:** استخدام وثائق Flutter الرسمية (مبسطة حالياً) لتحسين دقة الإجابات.

## المتطلبات
1. **Groq API Key:** للحصول على ذكاء اصطناعي سريع.
2. **Meta WhatsApp Business API:** (Token, Phone Number ID).
3. **Render Account:** للاستضافة.

## الإعداد
1. قم بنسخ الملف `.env.example` إلى `.env` واملأ البيانات المطلوبة.
2. قم بتثبيت المكتبات:
   ```bash
   pip install -r requirements.txt
   ```
3. قم بتهيئة قاعدة البيانات وإضافة البيانات التجريبية عن طريق طلب الرابط:
   `https://your-app-url.render.com/seed`

## الجداول المستخدمة
1. `Students`: بيانات الطالب ونقاطه.
2. `Lessons`: محتوى الدروس.
3. `Student_Progress`: تتبع تقدم الطالب.
4. `Quizzes`: الأسئلة المرتبطة بالدروس.

## الاستخدام
- أرسل رسالة للبوت للترحيب.
- اطلب **"درس اليوم"** للحصول على الدرس التالي.
- اطلب **"اختبار تحديد مستوى"** لتقييم مستواك.
